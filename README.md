## Как загрузить на GitHub
1. Создай новый репозиторий.
2. Нажми **Add file** → **Upload files**.
3. Загрузи все файлы из этого архива сразу.
4. Открой **Settings** → **Pages**.
5. В Source выбери **Deploy from a branch**.
6. В Branch выбери **main** и **/root**.
7. Сохрани.
8. Через минуту открой ссылку GitHub Pages.
